2D Maze Game

Project Overview

The 2D Maze Game is an interactive puzzle game where players navigate a character through a labyrinth to reach the exit. The game challenges players’ problem-solving and spatial reasoning skills while providing an engaging and visually appealing experience. Players encounter obstacles, dead ends, and sometimes collectible items that make the maze more dynamic and fun.

Features

  • Player Navigation: Move the character using keyboard controls (arrow keys or WASD).
  • Maze Generation: Pre-designed or randomly generated 2D mazes of varying difficulty levels.
  • Obstacles & Challenges: Dead ends, traps, or moving obstacles to increase difficulty.
  • Scoring System: Track completion time or collect items for points.
  • Restart & Levels: Ability to restart the maze or progress through increasingly challenging levels.
  • Visual Feedback: Highlighted paths or animations for player movement.

Technologies Used

  • Python
  • CSS
  • Java swing

What I Learned

I improved my understanding of 2d game design, event handling in Java Swing, and basic game mechanics implementation.
